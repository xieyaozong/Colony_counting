# Documentation

This folder contains project-level notes that are safe to publish with the code.

| File | Contents |
| --- | --- |
| `experiment_summary.md` | Method summary, validation metrics, and design rationale |
| `method_design.md` | Inference-pipeline design decisions and post-processing rationale |
| `evaluation_metrics.md` | Count-level metrics and why detection mAP is not enough |
| `failure_cases.md` | Known failure modes and practical mitigation ideas |
| `mac_compatibility.md` | Notes for running the project on Linux and macOS |

Raw datasets, private reports, generated PDFs, and local experiment artifacts are
not stored here.
